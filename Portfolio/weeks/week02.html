DATA TYPES

• String • Symbol (The symbol primitive data type was only introduced in ES6.) • Number • Boolean • Undefined • Null Any value that is not one of the primitive data types listed above is an object. These include arrays, functions and object literals.

JavaScript has a special operator called typeof for finding out the type of a value. typeof 'hello'<< 'string' typeof 10<< 'number' typeof true<< 'boolean' typeof { ninja: 'turtle' }<< 'object' typeof [ 1, 2, 3 ]<< 'object' (All non-primitive data types return a type of 'object' .)

ARRAYS

Ordered list of values. It can have other arrays inside and it is an object. Pop, Push, Shift, and Unshift: Pop: removes the last item of the array. Push: Appens new item to the last of the array. Shift: Removes the first item of the array. Unshift: Appens new item to the first place of the array. Concat: merges arrays. Join: Turns the array in string. Reverse: Reverses the order of the array. Sort: Sorts the array, alphabetically.
